# Silver-pulse

Static web app scaffold for a six-screen onboarding flow built with plain HTML, CSS, and vanilla JavaScript.

## Project Structure

- `index.html` - single-page app shell
- `styles.css` - responsive layout and base styling
- `script.js` - onboarding state, rendering, and navigation

## Getting Started

Open `index.html` in a browser to preview the current onboarding shell.

## Status

This repository currently contains the navigation architecture only. Final visuals and product-specific content can be layered onto the existing structure.
